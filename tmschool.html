Turkmen School Website

    • Function:
      • I developed a website for Turkmen School that serves as a resource to share the school’s mission and provide contact information for interested individuals.
      • I also integrated a chatbot system into the website, making it interactive and capable of answering user queries with context-driven responses.
    • Impact:
      • The website improved accessibility to information about the organization, allowing more people to learn about its mission and connect with it effectively.
      • The chatbot enhanced the user experience by providing quick and relevant answers to questions, making the website more engaging and useful.
    • Challenges:
      • One of the main challenges I faced was learning website development from scratch, which involved understanding HTML, CSS, and JavaScript.
      • Implementing OpenAI’s Embedding API and ChatGPT API for the chatbot system was also challenging, as it required grasping complex concepts and processes.
    • Solutions:
      • I addressed the challenges by engaging in self-guided learning and exploring various resources to understand web development tools and programming languages.
      • I broke down content into smaller chunks and used OpenAI’s Embedding API to create numerical embeddings that captured the meaning of the text.
      • Through testing and iterative development, I refined the chatbot system to ensure its responses were accurate and relevant to user queries.
    • Takeaways:
      • This project taught me the importance of self-learning and adaptability when tackling new challenges, especially in technology.
      • It also reinforced the value of combining creativity with technical skills to develop practical, user-focused solutions.
